## LearnTrack – Core Java Console Application

### Overview

LearnTrack is a menu-driven console application built using **Core Java**. It helps manage Students, Courses, and Enrollments using clean layered architecture.

### Features

* Student Management (Add, View, Search)
* Course Management (Add, View, Activate/Deactivate, Search)
* Enrollment Management (Enroll Student, View Enrollments)
* In-memory Repository (ArrayList)
* Custom Exception Handling
* Nested Menus

### Technologies Used

* Core Java (No Java 8 features)
* OOP Principles
* Collections (ArrayList) 

### Architecture

UI → Service → Repository → Entity

### How to Run

1. Clone the repository:- git clone https://github.com/wandreanurag/GitHub-leaner-repo.git
2. Open the project in any Java IDE (IntelliJ / Eclipse)
Navigate to:-com.airtribe.learntrack.ui.LearnerTrackerMain
Run LearnerTrackerMain.java
